## 🎯 **Project Overview**
**AI-Powered Student Information ChatBot** is a modern, responsive web application built for **BCA Final Year Project**. It provides instant answers to student queries about college information using intelligent keyword matching and natural language processing.

**Key Features:**
- 🤖 **Smart ChatBot** with 50+ responses
- 🔊 **Text-to-Speech** (Voice Responses)
- 👨‍💻 **Admin Dashboard** (CRUD Operations)
- 🎨 **Glassmorphism UI** (Dark Theme)
- 📱 **Fully Responsive** (Mobile/Desktop)
- ⚡ **Real-time Animations** & Effects
